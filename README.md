SmartPark is a simple Parking Management System website created for learning and demonstration purposes.

## Features
- **Parking Slot Availability**: View real-time (simulated) status of parking slots.
- **Booking System**: Easy-to-use booking interface for available slots.
- **Dummy Payment**: Simulated payment processing interface.
- **Admin Dashboard**: Visual overview of parking statistics and recent booking history.

## Project Structure
- `index.html`: Home page displaying slot availability.
- `booking.html`: Booking form.
- `payment.html`: Dummy payment gateway.
- `admin.html`: Dashboard for administrators.
- `css/`: Stylesheets.
- `js/`: JavaScript logic files.
- `images/`: Asset files.

## Technologies Used
- HTML5
- CSS3 (Custom properties, Flex, Grid)
- JavaScript (ES6+, LocalStorage for data persistence)

## How to Run
1. Clone the repository.
2. Open `index.html` in your web browser.
3. No backend server is required!

## Note
This project is frontend only. The payment system is a dummy UI, and data is stored in your browser's LocalStorage.
